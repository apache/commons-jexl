1. INTRODUCTION

The Jexl package implements a simple expression language engine. It borrows many ideas from the JSTL Expression Language. Liberally. Ok, it's an attempt at and idependent implementation of the JSTL EL with extensions.

2. DEPENDENCIES

The Jexl package is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • Jakarta Commons Logger (Version 1.0 or later)
  • Jakarta Velocity (Version 1.4 [current CVS] or later) - for introspection support
  • Jakarta Velocity DVSL (any version) - for documentation generation
  • dom4j (latest version) - for documentation generation
  • JUnit Testing Framework (Version 3.7 or later) - for unit tests only, not required for deployment
